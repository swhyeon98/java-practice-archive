# java-blackjack-precourse

블랙잭 미션은 딜러와 플레이어 중 카드 합이 21 또는 21에 가장 가까운 숫자를 가지는 쪽이 이기는 게임이다.

우아한테크코스 프리코스의 과제로,
**객체를 적절히 분리하고 협력시키는 연습**과 **단위 테스트 기반 구현**을 목표로 한다.

---

## 학습 목표

- 관련된 책임을 묶어 클래스로 분리하고, 객체들이 협력하여 하나의 기능을 완성하도록 설계한다.
- 핵심 도메인(카드, 유저 등)에 대한 단위 테스트를 작성하여 의도한 대로 동작하는지 확인한다.
- 프리코스 공통 피드백을 바탕으로,
    - 메서드 길이(10라인) 및 들여쓰기 depth 제한(2까지 허용)
    - else / switch 미사용
    - 입출력을 담당하는 클래스를 별도로 구현
    - 예외 처리 및 에러 메시지 일관성 등을 최대한 반영한다.
